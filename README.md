# Emergency Response Console

`waterdamagepromolab-mvp` has been reworked from a generic promo generator into a more practical restoration response console.

It now produces:

- dispatch and triage framing
- insurer-safe wording and homeowner talk tracks
- mitigation / drying / rebuild path options
- referral follow-up scripts for plumbers and property professionals
- 3 campaign angles with video hooks and storyboards
- a hero media concept via the `wuju.de5.net` image gateway

## Endpoints

- `GET /` response console UI
- `POST /api/console` full restoration workspace JSON
- `POST /api/promo-pack` compatibility response for the old shape
